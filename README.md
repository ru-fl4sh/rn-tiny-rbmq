# rn-tiny-rbmq

Tiny react native library for rabbit mq.

## Installation

```sh
npm install rn-tiny-rbmq
```

## Usage

```js
import RnTinyRbmq from "rn-tiny-rbmq";

// ...

const result = await RnTinyRbmq.multiply(3, 7);
```
